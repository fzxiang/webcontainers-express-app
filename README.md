    
# webcontainers-express-app

## Contributing

This is a webcontainers awesome,
click [here](https://fzxiang.github.io/webcontainers-express-app/) to see the demo.


Begin

```bash
pnpm i
pnpm run dev
```

Build

```bash
pnpm build
```

Generate, generate the server directly to json config
```bash
pnpm generate
```
